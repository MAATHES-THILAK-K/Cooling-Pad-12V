## 🧠 Design Overview

This cooling pad was designed with a balance of **performance, usability, and aesthetics**:

### 🔹 Ergonomics
- Elevated stand mechanism for improved posture
- Soft rubber grips for stability
- Compact and portable form factor

### 🔹 Cooling Performance
- 5-fan configuration for efficient heat dissipation
- Hexagonal vent design for optimized airflow
- 12V powered system for enhanced cooling capability

### 🔹 User-Centric Features
- USB-A & USB-C expansion ports
- Rotary encoder for quick control
- Transparent top panel concept for RGB aesthetics
